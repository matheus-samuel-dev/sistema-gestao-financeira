# 💰 Sistema de Gestão Financeira

Sistema Full Stack para gerenciamento financeiro pessoal, desenvolvido com **React**, **TypeScript**, **Spring Boot**, **PostgreSQL** e **Docker**.

A aplicação permite controlar receitas, despesas, metas financeiras e acompanhar indicadores através de um dashboard intuitivo.

---

## 🚀 Tecnologias

### Front-end
- React
- TypeScript
- Vite
- Material UI
- React Router

### Back-end
- Java 21
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- Hibernate
- Flyway

### Banco de Dados
- PostgreSQL

### DevOps
- Docker
- Docker Compose
- Nginx

---

## ✨ Funcionalidades

- ✅ Autenticação com JWT
- ✅ Dashboard financeiro
- ✅ Controle de receitas e despesas
- ✅ Gestão de metas financeiras
- ✅ Indicadores em tempo real
- ✅ Filtros e pesquisa
- ✅ API REST documentada com Swagger
- ✅ Banco versionado com Flyway
- ✅ Containerização com Docker

## ⚙️ Como executar

```bash
git clone https://github.com/matheus-samuel-dev/sistema-gestao-financeira.git

cd sistema-gestao-financeira

docker compose up --build
```

Após iniciar a aplicação:

- Front-end: http://localhost:3000
- API: http://localhost:8080
- Swagger: http://localhost:8080/swagger-ui.html
